자기소개와 관련된 웹 페이지를 만들 계획이야.
이와 관련된 1. 프로젝트 개요, 2. 프로젝트 상세, 3. 제약조건은 아래와 같아.


1. 목표
- 프로젝트명 : I AM
- 한 줄 정의 : 내가 좋아하는 주제를 매력적으로 소개하는 웹 서비스
- 목적 : 자신이 보여주고 싶은 내용을 AI를 활용하여 화면을 구현

2. 프로젝트 상세
- 주제
    - 1. 포트폴리오
        - 현재 올리는 PDF 파일의 첫번째 페이지의 내용을 기준으로 해서, 해당 내용을 웹 페이지 형태로 게시하고 싶음.
    - 2. 나에 대한 소개
        - 나의 취미 생활(기타 연주)을 다른 사람들에게 공유
        - 기타 연주하는 사진(추후 업로드 예정)과 동영상을 보여주고 싶음

3. 제약조건
- 1. 단일 페이지 구성
- 2. 콘텐츠 (기본 상호작용 : 버튼 클릭 리액션, 팝업 창 등) 구현
- 3. (중요) 바닐라 JS 활용 : HTML + CSS + JS 구조로 구현 (별도 프레임워크 및 빌드 도구 없이)
